Hangs up the connection to the mobile_image_mounter service. This functions has to be called before freeing up a mobile_image_mounter instance. If not, errors appear in the device's syslog.

```csharp
public MobileImageMounterError mobile_image_mounter_hangup(MobileImageMounterClientHandle client)
```

| parameter | description |
| --- | --- |
| client | The client to hang up |

## Return Value

MOBILE_IMAGE_MOUNTER_E_SUCCESS on success, MOBILE_IMAGE_MOUNTER_E_INVALID_ARG if client is invalid, or another error code otherwise.
