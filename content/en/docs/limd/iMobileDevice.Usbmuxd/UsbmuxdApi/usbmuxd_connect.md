```csharp
public virtual int usbmuxd_connect(uint handle, ushort tcpPort)
```

| parameter | description |

| handle | returned in the usbmux_device_info_t structure via usbmuxd_get_device() or usbmuxd_get_device_list(). |
| tcp_port | TCP port number on device, in range 0-65535. common values are 62078 for lockdown, and 22 for SSH. |
