```csharp
public SyslogRelayError syslog_relay_client_new(iDeviceHandle device, 
    LockdownServiceDescriptorHandle service, out SyslogRelayClientHandle client)
```

| parameter | description |
| --- | --- |

| service | The service descriptor returned by lockdownd_start_service. |
