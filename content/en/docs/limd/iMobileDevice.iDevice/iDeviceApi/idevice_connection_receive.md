Receive data from a device via the given connection. This function is like idevice_connection_receive_timeout, but with a predefined reasonable timeout.

```csharp
public virtual iDeviceError idevice_connection_receive(iDeviceConnectionHandle connection, 
    byte[] data, uint len, ref uint recvBytes)
```

| parameter | description |
| --- | --- |
| connection | The connection to receive data from. |
| data | Buffer that will be filled with the received data. This buffer has to be large enough to hold len bytes. |
| len | Buffer size or number of bytes to receive. |
| recv_bytes | Number of bytes actually received. |

## Return Value

IDEVICE_E_SUCCESS if ok, otherwise an error code.
