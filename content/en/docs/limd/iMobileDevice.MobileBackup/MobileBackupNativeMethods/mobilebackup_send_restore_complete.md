Tells the device that the restore process is complete and waits for the device to close the connection. After that, the device should reboot.

```csharp
public static MobileBackupError mobilebackup_send_restore_complete(MobileBackupClientHandle client)
```

| parameter | description |
| --- | --- |
| client | The connected MobileBackup client to use. |

## Return Value

MOBILEBACKUP_E_SUCCESS on success, MOBILEBACKUP_E_INVALID_ARG if client is invalid, MOBILEBACKUP_E_PLIST_ERROR if the received disconnect message plist is invalid, or MOBILEBACKUP_E_MUX_ERROR if a communication error occurs.
